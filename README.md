# IPL 2026 Mega Auction Data Analysis 🏏

An end-to-end data analysis project built using Python, Pandas, and NumPy. This project explores IPL 2026 Mega Auction data to uncover insights about player valuations, team spending, auction trends, and hidden gems.

## Features
- Data Cleaning & Preprocessing
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Team Spending Analysis
- Player Valuation Insights
- Hidden Gem Detection
- Interactive Portfolio Website

## Technologies Used
- Python
- Pandas
- NumPy
- HTML

## Project Highlights
- 244 Players Analyzed
- 175 Players Sold
- ₹644 Cr Total Auction Spending
- 10 IPL Franchises
